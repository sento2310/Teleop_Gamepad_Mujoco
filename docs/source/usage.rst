Usage Guide
===========

Basic Usage
-----------

.. code-block:: bash

   # Run with specific robot
   python gamepad_control.py panda
   
   # Interactive robot selection
   python gamepad_control.py

Controls
--------

Panda/UR5 Controls:
- Right Stick Y: Forward/Backward
- Left Stick X: Left/Right
- Left Stick Y: Up/Down
- A Button: Toggle Gripper
- START: Exit

SO100 Controls:
- Left Stick X: Rotation
- Left/Right Stick Y: Translation
- Shoulder Buttons: Wrist control
